Responsive Landing Page
Overview

This project is a modern and fully responsive landing page developed using HTML, CSS, and JavaScript. It features an interactive navigation menu with scroll-based styling changes, hover animations, smooth scrolling, and a mobile-friendly design.

Features
Fixed navigation bar visible on all sections
Navigation bar changes appearance on scroll
Interactive hover effects on menu items
Smooth scrolling between sections
Responsive design for desktop, tablet, and mobile devices
Modern UI with professional color scheme
Service and portfolio sections
Contact form section
Animated elements and transitions
Optimized user experience
Technologies Used
HTML5
CSS3
JavaScript (ES6)
