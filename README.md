# Responsive Landing Page | SkillCraft Technology Task 1

## SkillCraft Technology Web Development Internship - Task 1

This project is a modern, interactive, and fully responsive landing page developed as part of the SkillCraft Technology Web Development Internship Program.

Built using HTML5, CSS3, and Vanilla JavaScript, the website demonstrates modern frontend development practices including responsive layouts, interactive navigation, smooth scrolling, scroll-triggered animations, and a professional user interface.

---

## ✨ Features

### Fixed Interactive Navigation

* Fixed navigation bar visible across all sections.
* Navigation style changes dynamically on scroll.
* Smooth hover animations and active link highlighting.
* Responsive hamburger menu for mobile devices.
* Smooth scrolling navigation experience.

### Hero Section

* Eye-catching hero banner with modern typography.
* Professional call-to-action button.
* Responsive layout optimized for all screen sizes.
* Gradient and glassmorphism-inspired design elements.

### About Section

* Brief introduction section.
* Modern content layout.
* Responsive design with smooth animations.

### Services Section

* Responsive service cards.
* Hover effects and interactive transitions.
* Clean and modern card-based UI.

### Portfolio Section

* Showcase of featured projects.
* Responsive image layouts.
* Smooth hover interactions.

### Contact Section

* User-friendly contact form.
* Clean and accessible design.
* Responsive input fields and buttons.

### Animations & Effects

* Smooth scrolling behavior.
* Scroll-based navbar transitions.
* Hover animations on navigation links and buttons.
* Fade-in effects using JavaScript.
* Modern glassmorphism and shadow effects.

## 🚀 Installation & Running Locally

### Step 1: Clone Repository

```bash
git clone https://github.com/mangalawarsrihith07/SRIHITH-MANGALAWAR-SCT_WD_T1.git
```

### Step 2: Open Project Folder

Navigate to the downloaded project directory.

### Step 3: Run the Application

#### Option A: Direct Browser Launch

Open:

```text
index.html
```

in any modern browser such as:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

#### Option B: VS Code Live Server

1. Install the Live Server extension.
2. Open the project in VS Code.
3. Click **Go Live**.

#### Option C: Python Local Server

```bash
python -m http.server 8000
```

Open:

```text
http://localhost:8000
```

in your browser.
